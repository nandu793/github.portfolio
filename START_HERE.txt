🔌 API ENDPOINTS (20+)
═════════════════════════════════════════════════════════════════════════════

Authentication (7 endpoints)
  POST   /api/auth/register              Register with email
  POST   /api/auth/login                 Login with email/password
  POST   /api/auth/send-otp              Send OTP to phone
  POST   /api/auth/verify-otp            Verify OTP and login
  GET    /api/auth/google                Google OAuth
  GET    /api/auth/me                    Get current user
  PUT    /api/auth/profile               Update profile

Songs (7 endpoints)
  GET    /api/songs                      Get all songs
  GET    /api/songs/trending             Get trending songs
  GET    /api/songs/:id                  Get song details
  GET    /api/songs/stream/:id           Stream audio file
  GET    /api/songs/search?q=query       Search songs
  POST   /api/songs/:id/favorite         Add to favorites
  DELETE /api/songs/:id/favorite         Remove from favorites

Playlists (7 endpoints)
  POST   /api/playlists                  Create playlist
  GET    /api/playlists/user/my-playlists Get user playlists
  GET    /api/playlists/:id              Get playlist details
  GET    /api/playlists/public           Get public playlists
  POST   /api/playlists/:id/add-song     Add song to playlist
  DELETE /api/playlists/:id/remove-song  Remove song
  DELETE /api/playlists/:id              Delete playlist


🛠️ TECHNOLOGY STACK
═════════════════════════════════════════════════════════════════════════════

Backend:
  • Node.js - JavaScript runtime
  • Express.js - Web framework
  • MongoDB - NoSQL database
  • JWT - Authentication tokens

  • CORS - Cross-origin requests

Frontend:

  • CSS3 - Responsive design (Flexbox/Grid)
  • Vanilla JavaScript - No frameworks
  • Fetch API - HTTP requests
  • LocalStorage - Client-side data

Security:
  • Bcrypt hashing (10 rounds)
  • JWT tokens (7-day expiry)
  • Input validation
  • Error handling
  • CORS protection

🐛 QUICK TROUBLESHOOTING
═════════════════════════════════════════════════════════════════════════════

Problem                  | Solution
───────────────────────┼──────────────────────────────────────
Backend won't start     | Check if port 5000 is free, check .env
MongoDB connection err  | Ensure MongoDB is running
CORS error              | Verify backend on localhost:5000
Audio won't play        | Check database has songs, run seed.js
Login fails             | Try admin account, reset DB with seed.js
Port already in use     | Change PORT in .env to 5001
Can't find .env         | It's already created in backend/
